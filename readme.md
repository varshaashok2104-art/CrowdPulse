## Demo Flow

### Step 1: Idea Input
User enters a simple idea:
> **"AI-powered study planner for engineering students"**

### Step 2: AI Executive Team Activates
Our multi-agent system starts working in parallel:

- **🧑‍💼 CEO Agent** — Business viability & strategy
- **📊 Market Research Agent** — Market size, trends & opportunity
- **⚔️ Competitor Agent** — Existing products and market gaps
- **🛠️ Product Agent** — MVP features & recommended tech stack
- **💰 CFO Agent** — Revenue model, pricing & financials
- **📢 Marketing Agent** — Go-to-market strategy & user acquisition

### Step 3: The Killer Feature — Boardroom Debate ⭐
Instead of static text, you get a **live, animated boardroom debate** featuring **Harry Potter characters** in **Times Square** with voice narration.

You’ll see characters like Harry Potter, Hermione, Ron, and Dumbledore passionately discussing your idea — agreeing, disagreeing, roasting weak points, and refining the strategy in real time.

This is the moment judges remember after seeing 30+ AI projects.

## 🛠️ Tech Stack (Built in 24 Hours)

**Frontend:**
- React + Vite
- Tailwind CSS
- Motion (Framer Motion) for smooth animations

**Backend:**
- Node.js + Express
- Groq API (Llama 3 70B)

**Key Highlights:**
- No complex agent frameworks
- Real-time voice narration using browser TTS
- Dramatic Times Square boardroom UI with Harry Potter avatars
- Fully functional in under 24 hours
